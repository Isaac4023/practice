# Blog Personal - Manuel Isaac Delgado López

Este proyecto es una página web personal tipo "Portfolio/Blog" desarrollada como parte de mis prácticas de desarrollo web en **Dev.f**. El sitio funciona como una tarjeta de presentación digital donde comparto información sobre mí, mis intereses y formas de contacto.

##  Características

* **Barra de Navegación:** Enlaces de acceso rápido a las secciones principales.
* **Sección "Sobre Mí":** Presentación personal, fotografía y descripción profesional.
* **Listas de Intereses:** Pasatiempos y redes sociales organizados.
* **Formulario de Contacto:** Interfaz para envío de mensajes y datos de contacto.
* **Diseño Responsivo:** Estructura básica adaptable (HTML5).

##  Tecnologías Utilizadas

* **HTML5:** Estructura semántica.
* **CSS3:** Estilizado visual (vinculado a `src/css/styles.css`).

##  Tecnologías Utilizadas
* En unfuturo me gustaria agregar un diseño mas vistoso visualmente ademas de agregar mas campos acerca de mi experiencia laboral y escolar
